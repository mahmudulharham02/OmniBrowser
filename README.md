A modern, privacy-minded Android browser inspired by Brave. Lightweight, built with Kotlin and Jetpack Compose.

## Features

- Modern UI built with Jetpack Compose
- Fast and lightweight browsing experience
- Privacy-forward defaults
- Customizable (tabs, homepage, settings)

> Note: This project is an early-stage implementation — consider it experimental until otherwise noted.

## Getting Started

Requirements:
- Android Studio Flamingo or later
- Android SDK (API 26+ recommended)
- Kotlin

To run locally:

1. Clone the repo

   git clone https://github.com/mahmudulharham02/OmniBrowser.git
2. Open the project in Android Studio
3. Let Gradle sync and build
4. Run on an emulator or device
